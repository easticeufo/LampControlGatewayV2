Directive: esp-click

<tag esp-click="" ...>
Synopsis
<esp-click="URL">
Description

Similar to the href attribute, the esp-click attribute redirects the client to a new URL. However, it only redirects if the authorized user has the required abilities to access the route. The abilities are registered when the controller routes are declared. If the user does not have the required abilities, the message "Insufficient Privilege" is displayed in the feedback area. Otherwise the user is redirected to the requested URL.

Attribute Values
url: String Target URL.
Example
<span esp-click="/remove.html"></span>
