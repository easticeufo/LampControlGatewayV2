Directive: esp-input

<esp-input ...>
Synopsis
<esp-input type="TYPE" ng-model="NAME" format="FORMAT" label="LABEL">
Description

The esp-input generates an appropriate input element based on the type of the input field as determined from the database schema. The esp-input directive can generate "input", "textarea", "checkbox", "radio" and "select" form elements.

The esp-input directive supports the named attributes below. In addition, it will pass unknown attributes through to the input element.

If a label is not specified, the name of the field is used.

Attributes
filter: String Angular filter to apply to the value before display. This is useful to format dates or numbers.
inputClass: String CSS style to use for the input.
label: String Label text for the input field.
labelClass: String CSS style to use for the label.
ng-model: String Name of the model field to input.
type: String Type of control to render, (checkbox, text, textarea, date, email, tel, url, search, color, number, range).
value: String Default input value.
Example
<esp-input ng-model="event.id"></esp-input>
<esp-input type="text" ng-model="event.date" filter="date:'yyyy-MM-dd HH:mm:ss'"></esp-input>
<esp-input type="textarea" ng-model="event.message" rows="6"></esp-input>
<esp-input ng-model="event.title"></esp-input>
