Filter: esp-titlecase

{{ | titlecase}}
Synopsis
titlecase
Description

The esp-titlecase filter transforms the input so that each word begins with a capital letter. It may be used in Angular expressions as a piped filter.

Attributes
None
Example
<span>{{header | titlecase}}</span>
