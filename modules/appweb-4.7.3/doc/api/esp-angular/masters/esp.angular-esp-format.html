Filter: esp-format

{{ | format }}
Synopsis
format
Description

The esp-format filter formats values according to the relevant data type specified in the supplied database schema. When ESP responds to requests, it supplies the response data and the associated database schema that describes the various field types. This schema information may be used to automatically format fields before display.

Attributes
None
Example
<span>{{date | format:schema:date}}</span>
