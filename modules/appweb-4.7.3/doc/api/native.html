Native APIs

The Appweb Native APIs consist of C language headers, function prototypes, structure definitions and defines. They are divided into one of these areas.

Area Description
Appweb Appweb Http API
ESP Embedded Server Pages API
EDI Embedded Database Interface API
Http Http Client and Server Library API
MPR Multithreaded Platform Runtime API

Stability Classifications

Rather than providing a blanket backwards compatibility guarantee, APIs are classified according to their stability. These Stability Classifications should assist developers in migrating to new versions of Appweb.
